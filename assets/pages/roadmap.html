Data Science - My Experience

Hello! I wanted to open the writing section with my experience learning about Data Science, I will separate in two years what I have been learning and maybe I can help someone to start in this beautiful world.

Okay, with nothing more to add let's get started.

First Year

- Programming

Programming in Data Science is very important since the rest of the things that you are going to learn will have to be put into practice in a Language.

I recommend that your first language be Python and that when you are more or less advanced with it, learn R.

- Relational Databases

Other of the most important bases is to learn SQL, have an introduction to it, learn about Normalization, Concepts of relational databases, DDL and DML statements among other things.

- Data Collection and Integration

This consists of learning basic concepts of ETL, having and knowing about data quality, making big data clusters or processing in Azure are some of the many things covered in this section.

- Mathematics

Mathematics of course cannot be missing, mathematics is a very important part of data science and some of the things I learned were Operations with Matrices, Notions of propositional logic and several more.

- Mathematical Analysis

Following the previous point but a higher level is the Mathematical Analysis, I consider it important to learn about Linear function, Derivative and Integral or Continuous and Discontinuous Functions to give an example.

- Analytical Models

The Analytical Model provides insight and context for understanding the data. It is best to learn about processing and ingest, Analysis services available, Relational and non-relational database-driven data: similarities and differences.

- Data Mining

Data mining is the process of discovering useful information from large data sets. Use mathematical analysis to deduce patterns and trends that exist in the data, Analyze the quality of the data, Its concepts, Visualization (general concepts, techniques and good practices) or Applied Statistics.

- Data Mining Oriented Programming

Followed by Data Mining is learning about Libraries Oriented to programming on Data Mining, File Management, Access to WEB Services, Development of desktop applications are some of the things that I highlight.

Second Year

- Machine Learning

I love this part, machine learning is fantastic and to learn it I recommend starting with the Types of machine learning, Regression: linear, polynomial, logistic and decision limit are among the first things you should know.

Once you have learned the basics, it will be time to go further and learn about concepts of Deep Learning, Natural Language Processing, Neural Networks or Image Recognition.

- Non-Relational Database

This is easy, you basically learn NoSQL, CRUD operations, aggregation frameworks, simple index creation and replication.

- Statistic and Probability

Another of the most necessary things is Statistics together with Probability, such as the Binomial, Poisson and Normal Distribution. Contingency table and Venn diagram, Descriptive Statistics etc...

- Solution Architecture

This covers Cloud Architectures, Security Policies, Data Backup and Fault Tolerance, Big Data Oriented Architecture, Security Certifications and Compliance.

- Data Visualization

Learning and mastering data visualization tools such as Tableau or Power BI is an essential requirement today.

- Computer Law

I learned about computer law at the university, more than anything about Cybercrime, Protection of Personal Data, Copyright and everything that entails computer law
